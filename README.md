go-ansi 
=============

Installing
-------
```shell
    $ goinstall github.com/str1ngs/go-ansi/color
```
Example
-------
```go
    package main

    import (
        "fmt"
        color "github.com/str1ngs/go-ansi/color"
    )

    func main() {
        fmt.Printf("%s",color.Blue("techno color\n"))
    }
```
