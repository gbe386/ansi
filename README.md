go-ansi 
=============

Installing
-------
```shell
    $ goinstall github.com/str1ngs/go-ansi/color
```
Example
-------
```go
    package main

    import (
        "fmt"
        . "github.com/str1ngs/go-ansi/color"
    )

    func main() {
        fmt.Printf("%s\n", Blue("techno color"))
    }
```
